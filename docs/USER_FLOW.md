# Առաջադրանքի ստեղծման հոսք (User Flow)

## Քայլ առ քայլ պրոցես

### 1️⃣ Մուտք

Օգտատերը գնում է `/all-services` էջ և ընտրում է ենթակատեգորիա։

**Օրինակ:**
- Կատեգորիա: "Տրանսպորտի վերանորոգում"
- Ենթակատեգորիա: "Ավտոլվացում"

**Նավիգացիա:** 
```
/create-task/car-wash?category=auto-repair
```

---

### 2️⃣ Քայլ 1: Հիմնական տեղեկություններ

**Դաշտեր:**
- Վերնագիր (min 10, max 100 նիշ) - պարտադիր
- Կատեգորիա dropdown - pre-filled
- Ենթակատեգորիա dropdown - pre-filled

**Validation:**
- Բոլոր դաշտերը պարտադիր
- Real-time validation
- Error messages հայերեն

**UI:**
- Մեծ input box վերնագրի համար
- Երկու dropdown կողք կողքի
- Միայն "Հաջորդը →" կոճակ (full width, կանաչ)

---

### 3️⃣ Քայլ 2: Հատուկ հարցեր (Դինամիկ)

**Պայմանական քայլ:**
- Եթե ենթակատեգորիան ունի հատուկ հարցեր → ցույց տալ
- Եթե չունի → ավտոմատ անցնել Քայլ 3

**Օրինակ "Ավտոլվացում"-ի համար:**
- Ավտոմեքենայի տեսակ (radio buttons)
- Ծառայություններ (multiselect checkboxes)

**UI:**
- "Հետ" և "Հաջորդը" կոճակներ (40% / 60%)
- Հարցերը դինամիկորեն կառուցվում են `subcategory-forms.ts`-ից

---

### 4️⃣ Քայլ 3: Մանրամասն նկարագրություն

**Դաշտեր:**
- Textarea (min 50, max 2000 նիշ) - պարտադիր
- Character counter
- Խորհուրդների box

**UI:**
- 6 տող բարձրություն
- Real-time character count
- Info box խորհուրդներով

---

### 5️⃣ Քայլ 4: Գին և ժամկետ

**Դաշտեր:**
- Գնի տիպ (radio: Ֆիքսված / Բանակցային)
- Գումար (number input)
- Ժամկետ (dropdown)
- Կոնկրետ ամսաթիվ (պայմանական date picker)

**Validation:**
- Ֆիքսված գնի դեպքում գումարը պարտադիր է
- Կոնկրետ ամսաթվի դեպքում date picker-ը պարտադիր է

---

### 6️⃣ Քայլ 5: Տեղակայություն

**Դաշտեր:**
- Քաղաք dropdown - պարտադիր
- Հասցե text input - ըստ ցանկության
- Քարտեզ (placeholder - ապագայում)

**Քաղաքներ:**
Երևան, Գյումրի, Վանաձոր, և այլն

---

### 7️⃣ Քայլ 6: Նկարներ և կոնտակտ

**Դաշտեր:**
- Նկարներ upload (drag & drop, մինչև 5) - ըստ ցանկության
- Հեռախոս - պարտադիր
- Email - ըստ ցանկության
- Կապի եղանակներ (multiselect) - պարտադիր

**Image Upload:**
- Drag & drop support
- Preview thumbnails
- Հեռացման հնարավորություն
- Max 5 images

---

### 8️⃣ Քայլ 7: Ստուգում և հրապարակում

**Ցուցադրում:**
- Վերնագիր
- Կատեգորիա → Ենթակատեգորիա
- Գին և ժամկետ
- Տեղակայություն
- Հատուկ պարամետրեր
- Նկարագրություն
- Նկարներ (եթե կան)
- Կոնտակտ

**Գործողություններ:**
- ✓ Հրապարակել (primary button, full width)
- ← Հետ | Պահպանել սևագիր (40% / 60%)
- Խմբագրել (վերադառնալ քայլ 1)

---

## Auto-save

- Ավտոմատ պահպանում localStorage-ում
- 2 վրկ debounce
- Zustand persist middleware
- Վերականգնում բրաուզերի reload-ից հետո

## Mobile Responsive

- Mobile: կոճակներ կողք կողքի (35% / 65%)
- Desktop: կոճակներ կողք կողքի (40% / 60%)
- Adaptive font sizes
- Touch-friendly inputs
